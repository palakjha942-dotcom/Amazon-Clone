# Amazon Clone 🛒

A fully responsive Amazon Clone website built to replicate the core user interface and design features of the Amazon e-commerce platform.

## Features

* **Image Slider:** A dynamic and functional banner carousel on the homepage for highlighting deals and promotions.
* **Product Grids:** Well-structured and visually aligned layouts displaying multiple product categories cleanly.
* **Color-Corrected Footer:** An exact, fully styled multi-column navigation footer true to the original Amazon branding.
* **Responsive Design:** Optimized layout that adjusts smoothly across various device screens.

## Technologies Used

* **Frontend:** HTML5, CSS3
* **Scripting:** JavaScript (ES6+)

## Project Structure

* `index.html` - The main structure of the e-commerce store.
* `style.css` - Custom styling including responsive design and layout fixes.
* `first.js` - Logic driving the interactive features like the image slider.
* `images/` - Repository containing all UI icons and product media assets.
